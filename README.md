# Proseka Tools

A simple WinUI3 application with four sidebar navigation tabs.

## Project Structure

- **ProsekaToolsApp** - Main WinUI3 application
  - Four sidebar tabs using NavigationView:
    - Tab 1 - Home
    - Tab 2 - Documents
    - Tab 3 - Library
    - Tab 4 - Settings

## Requirements

- Windows 10 version 1809 (build 17763) or later
- .NET 8.0 SDK
- Visual Studio 2022 with the following workloads:
  - .NET Desktop Development
  - Universal Windows Platform development

## Building and Running

1. Open `ProsekaTools.sln` in Visual Studio 2022
2. Select your target platform (x64, x86, or ARM64)
3. Build and run the project (F5)

## Features

- Modern WinUI3 interface with Fluent Design
- Left sidebar navigation with four tabs
- Icon-based navigation items
- Responsive content area
- Clean and minimal design
